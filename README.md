# butian_sign_in
奇安信攻防社区 青龙自动签到脚本

## 使用
```shell
ql repo https://github.com/fengwenhua/butian_sign_in.git "butian.py" "" "sendNotify"
```

国内机器如下：

```shell
ql repo https://ghproxy.com/https://github.com/fengwenhua/butian_sign_in.git "butian.py" "" "sendNotify"
```
